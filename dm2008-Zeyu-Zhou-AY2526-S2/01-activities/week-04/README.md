# Week04
This week we learnt about class, which is like a template where it has it own variable and method (another name for function). Then when you need, can create different but somewhat similar objects with the template by passing in the values.
This is super useful, which I often see during game dev tutorial and always forget about it when I was coding... Another thing that is similar from Unity is encapsulation.

Another super important concept I learnt this week is the run logic of loop. I always thought loop run like this
Function start > loop 1 > out > function start > loop 2 > out > repeat until loop done. So function gets run as many times as loop needs
But it's actually like this
Function start > loop 1,2,3... until all done > out. So function run once, then loop runs till end.
