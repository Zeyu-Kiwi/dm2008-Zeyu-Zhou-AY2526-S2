# Week05
This week we learnt about collision. Ahhhh p5js collision is so troublesome compared to Unity. Unity has built-in function which I can just add 2D or 3D functions and it just work, it even can pass back collision information, and check collision enter, exit, and stay.
Collision for p5js is essentially check distance between objects and if the distance is smaller than the collision size, then collision happened. (for circle collider). Square collision is more complicated but the idea is somewhat the same.

We also learnt about changing shape's direction and speed with vector, setMag(), setHeading(). Vector needs a X and Y, which determines both the direction and speed.

Oh, and state change with boolean.
